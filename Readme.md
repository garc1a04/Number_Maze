**Grupo D**

## Integrantes
- **Bianca Oriá, Guilherme Garcia, Pedro Angelus**

## Linguagem

Java

## Como executar

### Pré-requisitos

- JDK 8 ou superior instalado

### Compilação

```bash
javac src/Main.java
```

### Execução

**Linux/macOS ou Windows (cmd):**
```bash
java -cp src Main < dados/entrada_do_problema.txt
```

**Windows (PowerShell):**
```powershell
Get-Content dados/entrada_do_problema.txt | java -cp src Main
```
